3.1. Control register

HW prefix: fmc_adc_core_ctl
HW address: 0x0
C prefix: CTL
C offset: 0x0

- - - - - - - -
23 22 21 20 19 18 17 16
- - TRIG_ERR FIFO_OF ADC_FSM[2:0] RIC
15 14 13 12 11 10 9 8
DBL_RATE ADC_OS[2:0] RANGE ADQ_M[1:0] RST
7 6 5 4 3 2 1 0
TRIGID[7:0]
  • TRIGID [read/write]: Trigger ID
    Is the trigger ID that starts the adquisition.
  • RST [read/write]: Reset ADC
    When this bit is set to 1 is done a Reset ADC and flags
  • ADQ_M [read/write]: Adquisition mode. On TimeStampMode the 24-bits less significative of harmony message with the triggerID are the time when adquisiton will be triggered.
    0: Continious adquisition
    1: On trigger
    2: TimeStamp
    3: no defined
  • RANGE [read/write]: Range
    ADC voltage range:
    0: +/-5V
    1: +/-10V (default)
  • ADC_OS [read/write]: Oversampling Mode
    0: No oversampling
    1: 2
    2: 4
    3: 8
    4: 16
    5: 32
    6: 64
    7: Invalid
  • DBL_RATE [read/write]: not Double Rate Input
    Sampling of adc (it is recomended to set to 1 only with adc_os=0, see datasheet)
    1: Sampling at 200kS/s
    0: Sampling at 400kS/s
  • RIC [read/write]: Read In Conversion
    when '1' it allows the possibility to read while conversion is running.
  • ADC_FSM [read-only]: State machine status
    States:
    0: ADC_RST
    1&5: IDLE
    2&6: inConversion
    3&7: reading
    8: illegal
  • FIFO_OF [read-only]: ADC FIFO overflow
    ADC FIFO has and overflow
  • TRIG_ERR [read-only]: Trigger error
    Trigger tried to send but adc was not ready.
