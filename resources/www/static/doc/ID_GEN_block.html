3.1. ID generator Block Control Register

HW address: 0x0

C offset: 0x0

RST ERR RDY CLKE[1:0] DSRC ETI MTRIG
7 6 5 4 3 2 1 0
TRIGGERID[7:0]
  • TRIGGERID [read/write]: Trigger ID
    if EnTrigID is set to 1, each time this ID is received a message is generated in Harmony BUS
  • MTRIG [write-only]: MTrigger
    Sends the Data Message when it is set.
  • ETI [read/write]: EnTrigID
    If it is set to 1, the data mesage is send in Harmony bus each time a Trigger ID is received.
  • DSRC [read/write]: DataSRC
    If it is set to 1, the data field of out message is the data received from last TriggerID message received.
  • CLKE [read/write]: CLKE
    The out message's timeStamp source is:
    * 0: SDB register TS_ID.TimeStamp.
    * 1: Internally generated.
    * 2: The last TriggerID received.
    * 3: Reserved.
  • RDY [read-only]: Ready
    this bit indicates that data is still not send to the Harmony Bus and an error can be generated if a new data request is generated.
  • ERR [read-only]: IDGEN Error
    This bits indicates that ID-Gen block has a error. The main reason for this is that a new message is required to be send when a previous message is still pending to be delivered.
  • RST [write-only]: ID-GEN Error Reset
    Resets the IDGEN Error and the Ready fields. This may produce a lost of a message pending to be send.
