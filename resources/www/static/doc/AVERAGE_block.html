3.1. Average Block Control Register

HW address: 0x0

C offset: 0x0

AOF ENA[1:0] PRE_DIV[4:0]
  • PRE_DIV [read/write]: PRE_DIV
    This bit indicates the number of bits that output is shifted respect to acomulator register.
  • ENA [read/write]: Enable Output
    These bits indicates, when output data will be generated.
    00: Data is send for each ID_IP or ID_IN is received.
    01: Data is send after receive a positive input.
    10: Data is send after receive a negative input.
    11: Data is send after receive both positive and negative inputs.
  • AOF [read-only]: Acumulator overflow
    This bit indicates that FIFO is full and it is ready to send output data.
  • OOF [read-only]: Output overflow
    This bit indicates that acoumulator has a bigger value than the output send.
  • WB_RST [write-only]: Average Reset
    Resets the acomulator registers to 0 and clear the flags Acumulator overflow and Output overflow

  • ID_IP [read/write]: Input ID POSITIVE
    This is the ID data that will be added at the acomulator. If it is set to 0 no data is added.
  • ID_IN [read/write]: Input ID NEGATIVE
    This is the ID data that will be substracted at the acomulator. If it is set to 0 no data is sustracted.
  • ID_OUT [read/write]: Output ID
    This is the ID of the output data generated the Timestamp of Output is the timestamp of the last input. If it is set to 0 no data is generated.
