3. Register description

3.1. Digital input output Setup

HW prefix: cnt_block_dio_stp
HW address: 0x0
C prefix: DIO_STP
C offset: 0x0
31 30 29 28 27 26 25 24
- RST HO IO[12:8]
23 22 21 20 19 18 17 16
IO[7:0]
15 14 13 12 11 10 9 8
- - - IOP[12:8]
7 6 5 4 3 2 1 0
IOP[7:0]
  • IOP [read/write]: Digital input Polarity
    If it is 1 the signal is polarity is changed. The bit 0 refers to DIO1 and bit 4 to DIFF_IO_1.
  • IO [read/write]: Setup Digital input output.
    Sets if GPIO is an input(1) or output(0). The bit 0 refers to DIO1 and bit 4 to DIFF_IO_1.
  • HO [read-only]: harmony overflow
    Indicates that and overflow was produced trying to send a harmony message.
  • RST [write-only]: Reset
    Reset the flags of em2_digital_io.

D_MASK[7:0]
  • D_MASK [read/write]: Digital input output Mask
    Digital output mask, first the DIO1..4 and then DiffIO1..9. If this register is set to '1' the value of next value written at DIO_V, it is written at the output, and the output logic is disabled. DO NOT WRITE IN NORMAL OPERATION, WRITE ONLY FOR TEST PROCEDURES!!
  • DIO_V [read/write]: Digital input output Value
    Digital input output Value, first the DIO1..4 and then DiffIO1..9. The output value will only work if D_Mask is set to '1'. DO NOT WRITE IN NORMAL OPERATION, WRITE ONLY FOR TEST PROCEDURES!!

3.3. Harmony counter disable setup

HW prefix: cnt_block_dis_stp
HW address: 0x2
C prefix: DIS_STP
C offset: 0x8
31 30 29 28 27 26 25 24
- - - - WD2 DIS2 DIS2_M[1:0]
23 22 21 20 19 18 17 16
DIS2_ID[7:0]
15 14 13 12 11 10 9 8
- - - - WD1 DIS1 DIS1_M[1:0]
7 6 5 4 3 2 1 0
DIS1_ID[7:0]
  • DIS1_ID [read/write]: Disable1 ID
    ID of the message that generates the disable signal
  • DIS1_M [read/write]: Disable1 Mode
    Functionality mode of Harmony disable 1:
    00: Each ID recieved changes the disable signal.
    01: the ID recieved always enables the disable
    10: the ID recieved always disables the disable
    11: Manual control
  • DIS1 [read-only]: disable1
    Indicates the state of Harmony Disable line1.
  • WD1 [read/write]: Write DIS1
    When disable Mode is '11', this bit it is changes the disable state. '1' enable and '0' disable.
  • DIS2_ID [read/write]: Disable2 ID
    ID of the message that generates the disable signal
  • DIS2_M [read/write]: Disable2 Mode
    Functionality mode of Harmony disable 1:
    00: Each ID recieved changes the disable signal.
    01: the ID recieved always enables the disable
    10: the ID recieved always disables the disable
    11: Manual control
  • DIS2 [read-only]: disable2
    Indicates the state of Harmony Disable line2.
  • WD2 [read/write]: Write DIS1
    When disable Mode is '11', this bit it changes the disable state. '1' enable and '0' disable.
